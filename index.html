My Projects

Sentiment Analysis, Product Rating Web Application (C#)


Opminr is a sentiment analysis engine that was built as my final year University Computer Science project.

It works by connecting to Twitter to gather Tweets which contain the user's input, performing sentiment analysis on the Tweets to mark them as positive, negative or neutral, analysing this data to provide the user with a rating out of 5 stars as well as lots of other useful information.

My final dissertation can be found in a .PDF file here.

Multiplayer Tower Defence Game (Java)


As part of my Team Project module during second year, I worked in a team of five to create a multiplayer game that was both competitive and cooperative. We came up with the idea of a multiplayer tower defence game.

My specific job in this project was to create a dedicated map editor using OpenGL and I also worked in a pair to create the Artificial Intelligence.

After we had completed a mock sales pitch to the lecturers running the course and also a representative from IBM we were awarded the IBM University Team Challenge for our teamwork through the project.

Work Management Application Design (Balsamiq)
