Email Servlet

The idea of this simple Java servlet is that the user can log into their GMail account and send an email. The reason as to why I developed this was because of my networking module that required an online servlet that is capable of sending emails.

I used the Twitter Bootstrap framework to create the simple form.

Falling Block Game

At the end of my first year at university, I was asked to create a simple block falling game for my module Software Workshop in Java. The idea of this game is to hit SPACE when the colour of the falling block lines up with one below. The game ends when the board is cleared. However miss a block and the bottom will raise by one.
